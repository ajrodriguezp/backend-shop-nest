# ALAN SHOP DB
1. Clonar el repositorio
2. ```npm i ```
3. Clonar el archivo ```.env.template``` y renombrar a .env
4. Cambiar las variables de entorno
5. Ejecutar el contenedor para levantar la base de datos

```
docker compose up -d
```

6. Ejecutar modo dev ```npm run start:dev```
